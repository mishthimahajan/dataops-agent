🚀 DataOps AI Environment
📌 Overview

An AI-powered environment that simulates real-world data cleaning and analytics workflows using natural language instructions.

✨ Features
💬 Chat-based UI
🎤 Voice input
🧠 NLP instruction parsing
📊 Before vs After comparison
📈 Reward visualization
🎯 Score tracking
🤖 Auto agent execution
🧠 Tasks
Easy → Clean missing values
Medium → Clean + type conversion
Hard → Full pipeline + analytics
⚙️ Tech Stack
Frontend: React + Chart.js
Backend: Flask
Deployment: Render + Vercel
🚀 How to Run
Backend
cd backend
py app.py
Frontend
cd frontend
npm run dev
